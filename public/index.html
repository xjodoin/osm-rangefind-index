Point Rangefind at this origin. The runtime fetches only the manifests and immutable packs needed to answer each query.

Search from JavaScript

Install rangefind, create the search engine, and query OpenStreetMap places directly from the browser or Node.

npm install rangefind

import { createSearch } from "rangefind";

const search = await createSearch({
  baseUrl: "https://osm.rangefind.dev/",
});

const result = await search.search({
  q: "1234 rue sainte-catherine",
  size: 5,
});
